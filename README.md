# QR Generator (Python)

Script simple para generar códigos QR a partir de una URL o texto.

## Requisitos
- Python 3.10+
- Dependencias: ver `requirements.txt`

## Uso
```bash
python generadorqr.py "https://example.com"
```

Genera un PNG con el QR en el directorio actual.

## Notas
- Este proyecto es público y no contiene secretos.
